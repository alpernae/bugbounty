# Overview

This SKILL file stores the side skills, workflows and methodologies that can be extracted from conversations guided by the main skill. It serves as a reference for identifying and developing additional skills that complement the main skill's objectives.

## Bug Bounty Hunter: h1-learning.SKILL.md

This skill focuses on learning from HackerOne reports using MCP to automate agent learning from valid bug bounty findings. It involves acccessing, parsing, and analyzing report content to extract key insights and remediation steps, which are then memorized for continuous improvement of agent skills in vulnerability detection and report generation.

1. Access HackerOne reports via MCP
2. Parse report content for key information
3. Analyze findings and remediation steps
4. Memorize insights in h1-learning-memory.instruction.md
5. Update agent skills with new knowledge
6. Validate and iterate on learning process
