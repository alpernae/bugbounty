# 0x04

The `0x04` directory contains resources for integrating AI-powered agents into your bug bounty workflow using Visual Studio Code (VS Code) Copilot. This section focuses on two key agents: **Astra** and **ReportAI**.

## Folder Structure

```
0x04/
├── README.md
└── VS-Code/
    ├── agents/
        └── Astra.agent.md
        └── ReportAI.agent.md
```

## Astra Agent

**Astra** is an AI assistant designed to help automate reconnaissance, vulnerability discovery, and code analysis tasks. It can:

- Suggest payloads and attack vectors based on context.
- Analyze code or HTTP responses for potential vulnerabilities.
- Provide recommendations for next steps during testing.

## ReportAI Agent

**ReportAI** streamlines the process of documenting and reporting vulnerabilities. It can:

- Generate structured vulnerability reports from your findings.
- Suggest remediation steps and references.
- Format reports according to common bug bounty program requirements.
