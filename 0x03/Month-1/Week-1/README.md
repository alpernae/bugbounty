## Week 1: Web Fundamentals Deep Dive (6 Days) - Detailed Plan

This week focuses on building a strong foundation in core web technologies: HTTP, HTML, CSS, and JavaScript. This is crucial for bug bounty hunting.  Learning Hours (LH) are estimated and can be adjusted based on your pace. Hunting Hours (HH) are not included in the first month as the focus is on learning.
