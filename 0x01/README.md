# 0x01

This folder contains bug bounty note templates and personal documentation for tracking bug bounty engagements.

## Contents
- **BBTNT.md**: Bug Bounty Testing Note Template for efficiently tracking progress, findings, and critical details during bug bounty programs.

## Usage
- Use `BBTNT.md` as a starting point for each new bug bounty program or engagement.
- Fill in the tables and sections as you progress through your testing.
